BusCraft
========

Prototype of BusCraft, a Python / PySide6 based UVM VIP generator with a plugin
architecture for multiple bus protocols.

Quick start:

    python -m venv .venv
    # Windows
    .venv\Scripts\activate
    # Linux/macOS
    source .venv/bin/activate

    pip install --upgrade pip
    pip install -r requirements.txt

    python -m buscraft.main
